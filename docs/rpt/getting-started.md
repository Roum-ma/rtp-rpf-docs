# Getting Started with RPT

## What is RPT

**RPT** is a modification for Minecraft Java Edition that adds some additional capabilities for item model creators in resource packs. It adds new model types, a variable system, and templates.

## Installation
 1) Install [RPF](/rpf/getting-started.md) as a mod.
 2) Install [RPT](https://modrinth.com/mod/rp-twks) as a mod.
 3) Done!

## How to Report a Bug
In the world of software, there are situations when it doesn't behave as intended. If you happen to notice that RPT stopped working as it might have been intended, you can reach out in the [Issues](https://github.com/Danrus1100/rpt/issues) section of the project repository, or in the project's [Discord](https://discord.gg/Fkp9Tqmm6F), I will always try to help you or resolve the issue as quickly as possible!
