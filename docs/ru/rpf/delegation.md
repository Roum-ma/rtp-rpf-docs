# Система делегирования

## О стратегиях выборки

в **RPF** есть 3 стратегии выборки: `rpf:v1`, `minecraft:vanilla` и `rpf:experemental`. Вы можете переключаться между ними, написав в чат `/rpf resolver set {id}`, а узнать текущий - `/rpf resolver`

### rpf:v1

Просматривает модели из всех наборов сверху вниз, прерывая цикл когда модель не делегировала, рендеря её

### rpf:experemental

Сначала тестирует все модели, а потом, судя по очкам, которые модель набрала (они набираются и вычитаются в зависимости от глубины модели и их типа) выбирает модель с самым большим балом и рендерит её

::: warning Обратите внимание
Стратегия названна "Эксперементальной", потому что она **НЕ** стабильна! В будущих обновлениях она так же может быть переименована
:::

### minecraft:vanilla

Стандартная статегия выборки, написанная Mojang
